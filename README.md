 IMDB Movie Review Sentiment Analysis

 Overview

This project uses machine learning and natural language processing (NLP) to classify IMDB movie reviews as positive or negative. The model is trained on a labeled dataset of movie reviews and learns to predict sentiment based on the text content.

---

 Purpose

The goal of this project is to:

* Explore how sentiment analysis works
* Apply NLP techniques to real-world data
* Build and evaluate a classification model
* Gain experience with machine learning workflows

---

 Dataset

* **Source:** IMDB Movie Reviews Dataset
* **Size:** 50,000 reviews
* **Labels:** Positive / Negative
* Balanced dataset for accurate training
